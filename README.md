## おつむ弱者でもわかるたぐつーるの使い方
### その１
<img src="画像ども/説明に使う/おぼえろ.png" width="650"></br>
テンプレートに沿って入力！↓てんぷれ</br>
<sub>アビ番号(ふれあび製作者以外なら任意)　能力名</sub></br>

### その2
<img src="画像ども/説明に使う/おぼえろ2.png" width="650"></br>
能力に該当するタグにチェック！(タグの説明は後日書いておきますね)</br>
### その3
<img src="画像ども/説明に使う/おぼえろ3.png" width="650"></br>
全部整ったら生成を押す！そのままだと表示が崩れてるのでコピーボタンを押してどっかメモ帳かなんかに張り付ける！</br>
### 番外1
<img src="画像ども/説明に使う/おぼえろ4.png" width="650"></br>

<img src="画像ども/説明に使う/おぼえろ5.png" width="650"></br>
